# Zomato Restaurant Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Zomato Restaurant Dataset. The objective is to understand restaurant trends, customer preferences, ratings, cuisines, online delivery availability, and pricing patterns using Python and data visualization techniques.

## Dataset
Source: Kaggle - Zomato Restaurant Dataset

The dataset contains information about:
- Restaurant names
- Locations
- Cuisines
- Ratings
- Online delivery availability
- Average cost for two
- Votes and customer feedback

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Tasks
### 1. Data Cleaning
- Checked missing values
- Removed duplicates
- Verified data types
- Prepared data for analysis

### 2. Exploratory Data Analysis
Analyzed:
- Top cities with restaurants
- Most popular cuisines
- Average ratings by city
- Online delivery trends
- Cost and rating relationships

### 3. Data Visualization
Created:
- Bar Chart
- Histogram
- Pie Chart
- Line Chart
- Scatter Plot
- Heatmap

## Key Insights
1. Some cities have significantly more restaurants than others.
2. Certain cuisines dominate the restaurant market.
3. Online delivery is widely adopted in major cities.
4. Restaurant price does not always determine customer rating.
5. Customer preferences vary across locations.
